# News Sentiment and Market Dynamics — BTC / GLD

Predicting next-day volatility spikes in Bitcoin and Gold (GLD ETF) from
news sentiment, July–December 2024.

## Datasets

| File | Rows | Description |
|---|---|---|
| `normalized_news_july_dec_2024.csv` | ~21K | News headlines tagged `financial`, `political`, or both; 50+ sources |
| `bitcoin_prices_2024_4h.csv` | 1,098 | BTC OHLCV at 4-hour resolution |
| `GLD_ETF_Stock_Price_History_Converted.csv` | 128 | GLD ETF daily OHLCV |

## Pipeline

1. **Pseudo-label** a stratified 5K sample of headlines with zero-shot
   BART-MNLI using market-framed hypotheses.
2. **Filter** by confidence (≥ 0.65) — keeps ~2.9K high-quality labels.
3. **Fine-tune** FinancialBERT (bottom 4 encoder layers + embeddings
   frozen) for 4 epochs on the pseudo-labeled set.
4. **Score** all 21K headlines with the fine-tuned model.
5. **Aggregate** to daily sentiment features, split into three parallel
   streams: `all_*`, `fin_*` (financial-tagged), `pol_*` (political-tagged).
6. **Merge** with BTC / GLD daily OHLCV and train spike-prediction models
   (next step — see `finetune_financialbert.py` for stages 1–5).

Run:

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install --upgrade torch torchvision torchaudio
pip install transformers datasets scikit-learn pandas numpy accelerate
python finetune_financialbert.py
```

On an Apple Silicon M5 Pro / 48GB the full pipeline finishes in ~10 min
using MPS acceleration.

## Results — Fine-tuning Ablation

Held-out test set of 291 headlines (binary positive / negative, since
zero-shot pseudo-labeling produced < 1% confident "neutral" examples).

| Model | Accuracy | F1 (macro) |
|---|---|---|
| FinancialBERT (off-the-shelf) | 0.526 | 0.423 |
| **FinancialBERT (fine-tuned)** | **0.790** | **0.790** |

### Per-class breakdown (fine-tuned)

```
              precision    recall  f1-score   support
    negative      0.809     0.770     0.789       148
    positive      0.773     0.811     0.792       143
    accuracy                          0.790       291
   macro avg      0.791     0.791     0.790       291
```

### Confusion matrix (fine-tuned)

|          | pred neg | pred pos |
|----------|---------:|---------:|
| **true neg** | 114 | 34 |
| **true pos** | 27 | 116 |

The baseline collapses onto "negative" (138/148 true negatives caught,
only 15/143 true positives) because FinancialBERT's pre-training label
distribution is very different from the market-framed zero-shot
hypotheses used for pseudo-labeling. Fine-tuning aligns the head to the
target labeling scheme — this is a scheme-alignment gain, not a proof
of superhuman sentiment accuracy.
